== Basic Rails/Backbone.js App

Hey there, I'm posting this simple little app as reference for anyone learning Rails/Backbone, as I am.

Here are some things to be aware of if you're going to look at this code:

  1. After git clone'ing this repo, do the following to get the site running locally at http://localhost:3000
bundle install
rake db:migrate
rails server
  1. I'm using the HTML5 editablecontent div/span attribute to allow for in-place text editing, so if your browser doesn't support HTML5, it won't work
  2. I used this tutorial to build the backend
  3. You can go to http://localhost:3000/posts (and other posts-related resource URLs) to experience a non-Backbone version of the site
  4. I've turned off the csrf protection so that the AJAX PUT/POST calls will work without Rails denying the requests
